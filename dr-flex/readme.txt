Das offizielle Dr. Flex® Wordpress Plugin zur einfachen Einbindung des Dr. Flex® Buchungstools auf Ihrer Website.

== Description ==
Vergeben Sie Ihre Arzttermine direkt auf Ihrer WordPress Website. Das offizielle Dr. Flex Plug-in lässt sich schnell und einfach in Ihren Webauftritt integrieren. 

Der Patientenkalender erscheint auf Ihrer Praxiswebsite und kann kinderleicht konfiguriert werden, um vor dem Arzttermin alle für Sie relevanten Informationen abzufragen. So können Ihre Patient:innen in weniger als 30 Sek. Termine vereinbaren - ohne Registrierung oder Anmeldung. 
Alle Informationen werden DSGVO-konform an Ihre Praxissoftware übermittelt und dort angezeigt. Außerdem werden keine externen Skripte auf Ihre Website geladen, sodass keine Erwähnung von Dr. Flex im Cookie-Banner für Sie erforderlich ist.  
Das offizielle Terminvergabe-Tool bietet viele Features wie eine SMS-TAN-Verifizierung, Terminbestätigung via SMS, Schnittstellen zu 27+ Arztsoftware-Programmen und mehr - bereits ab 29€/Monat, ohne Vertragslaufzeit und jederzeit kündbar. 
Vereinbaren Sie Ihr erstes Beratungsgespräch mit uns!

== Installation ==
Nach dem kostenlosen Beratungsgespräch wird unser Team Ihnen einen persönlichen API-Key schicken. Um die Installation der Plug-ins durchführen zu können, geben Sie den API-Key in der Einstellungsmaske ein. Damit werden das Plug-in und Ihr Dr. Flex Praxis-Account sicher verknüpft. 
Anschließend können Sie das Buchungstool über einen WordPress Shortcode oder über die JavaScript-Funktion ‚javascript:toggleDrFlexAppointments()‘ schnell und unkompliziert direkt auf den gewünschten Seiten einbinden. Ist die Einbindung abgeschlossen, kann das Booking-Widget sofort auf Ihrer Webseite verwendet werden. Viel Spaß!
